# refrigerator
使用ionic混合开发技术开发的APP,能通过蓝牙控制冰箱
## 主要有以下四个界面：冰箱设备搜索页面、主页面、冰箱设置、侧边栏页面
安装包在app-debug.apk，描述文件在智能冰箱.docx
